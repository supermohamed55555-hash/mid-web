CSS Selectors

Targeting HTML elements with precision to create stunning designs and maintainable codebases.

What is a Selector?

A CSS selector is used to target HTML elements so you can apply styles to them. It is the first part of any CSS rule.
الـ Selector في CSS بيستخدم عشان تحدد العناصر اللي عايز تطبق عليها التصميم، وهو دا بيبقى أول جزء في أي قاعدة CSS بتكتبها.

Specificity Hierarchy

* Universal Selector Power: 0
div Element Selector Power: 1
.class Class Selector Power: 10
#id ID Selector Power: 100

Type Selector (Element)

Targets all elements of a specific type (tag name). Use it for global styles like general text or headers.
بيحدد كل العناصر اللي من نوع معين في الصفحة كلها (زي كل الـ p أو كل الـ h1).
p {
  color: blue;
}

Result: All <p> tags will turn blue.

Class Selector

Starts with a dot (.) and targets any element with that specific class. You can use a class multiple times in one page.
بيبدأ بنقطة (.) وبيحدد أي عنصر مدي له الكلاس دا. تقدر تستخدم الكلاس أكتر من مرة عادي في نفس الصفحة.
<p class="highlight">Hello</p>

.highlight {
  background: yellow;
}

Targeting Class within Element

You can target a class only when it is applied to a specific element type.
ممكن تحدد كلاس معين بس لو كان موجود داخل تـاج (tag) معين انت عايزه.
span.highlight {
  color: red;
}

Result: Only <span> with 'highlight' class will be affected.

ID Selector

Starts with # and targets a unique element. Each ID must be unique per page.
بيبدأ بعلامة الـ # وبيحدد عنصر واحد بس "بصمة". ممنوع تكرر نفس الـ ID مرتين في نفس الصفحة.
<h1 id="title">Hello</h1>

#title {
  color: green;
}

Multiple Classes

An element can have more than one class. You can target elements that have all specified classes.
العنصر ممكن يكون واخد كلاسين أو أكتر في نفس الوقت، وتقدر تستهدف العنصر اللي فيه الكلاسين بالملي.
<div class="box warning"></div>

.box.warning {
  border: 2px solid red;
}

Selector Lists

Apply the same style to multiple selectors by separating them with commas.
ممكن تطبق نفس الستايل على أكتر من عنصر مرة واحدة باستخدام الفاصلة بين الـ selectors.
h1, p {
  color: blue;
}

Universal Selector

The asterisk (*) selects all elements on the page. Usually used for resets.
النجمة (*) بتحدد كل العناصر اللي موجودة في الصفحة من أولها لآخرها (عادة بنستخدمها للـ Reset).
* {
  margin: 0;
  padding: 0;
}
