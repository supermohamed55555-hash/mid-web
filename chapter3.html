03

Lists, Tables & Forms

Structuring complex data and handling user interactions with advanced HTML elements.

HTML Lists

A list in HTML is used to group related items together in an organized way.
الـ List في HTML بتستخدم عشان تجمع عناصر مرتبطة ببعض بشكل منظم.

Unordered List <ul>

The <ul> element represents an unordered list. Items are displayed with bullets, and their order does not matter.
عنصر <ul> بيستخدم لعمل قائمة غير مرتبة، والعناصر بتظهر على شكل نقط (Bullets)، والترتيب مش مهم.
<ul>
  <li>Milk</li>
  <li>Cheese</li>
  <li>Bread</li>
</ul>

Ordered List <ol>

The <ol> element represents an ordered list. Items are displayed with numbers, and the order matters.
عنصر <ol> بيستخدم لعمل قائمة مرتبة، والعناصر بتظهر بأرقام والترتيب فيها مهم.
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

Nested & Mixed Lists

You can place a list inside another list. This is called nesting. You can also mix ordered and unordered lists together.
ممكن تحط قائمة جوه قائمة تانية، وده اسمه Nesting. وممكن كمان تدمج بين الـ <ul> والـ <ol> في نفس الوقت.
<ul>
  <li>Cheese
    <ul>
      <li>Blue cheese</li>
    </ul>
  </li>
</ul>

HTML Tables

A table is used to display structured data in rows and columns.
الـ Table بيستخدم لعرض البيانات في شكل صفوف وأعمدة منظمة.
Name Age Skill
Ahmed 20 HTML
Mona 22 CSS
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ahmed</td>
    <td>20</td>
  </tr>
</table>
<table>: Container.
<tr>: Table Row.
<th>: Header Cell (Bold).
<td>: Data Cell.
table: الجدول نفسه.
tr: الصف.
th: عنوان العمود (بيكون تقيل).
td: البيانات اللي جوه الخلية.

HTML Forms (Input Types + Label)

Forms collect user data using different input types. Common input types include text, email, password, and submit. The <label> element is used to describe the input field clearly and improve usability (clicking the label focuses the input).
الـ Forms بتجمع بيانات المستخدم باستخدام أنواع مختلفة من الحقول. من أشهر الأنواع: text و email و password و submit. وعنصر <label> بيستخدم لوصف الحقل بشكل واضح وتحسين سهولة الاستخدام (لما تضغط عليه الماوس بيروح للحقل فوراً).
<form>
  <label for="name">Name:</label>
  <input type="text" id="name">

  <label for="email">Email:</label>
  <input type="email" id="email">

  <label for="password">Password:</label>
  <input type="password" id="password">

  <input type="submit" value="Submit">
</form>
