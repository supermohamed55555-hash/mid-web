Advanced Text Styling

Going beyond basics: Interactivity, Web Fonts, and sophisticated layout controls.

What is Advanced Text Styling?

Advanced styling focuses on interactivity (links), custom resources (Web Fonts), and fine-tuning the reading experience.
الـ Advanced Styling بيهتم أكتر بالتفاعل مع المستخدم (زي اللينكات)، واستخدام مصادر خارجية زي خطوط جوجل، وتظبيط التفاصيل اللي بتخلي القراءة ممتعة.

Styling Links (States)

Links change their appearance based on user interaction: Normal, Hover, and Visited.
اللينكات في الـ CSS ليها حالات مختلفة حسب انت عملت إيه؛ سواء لسة شايف اللينك، أو واقف عليه بالماوس، أو ضغطت عليه قبل كدة.
NORMAL Click Me
HOVER (🔥) Hovered
VISITED Visited
a { color: blue; }
a:hover { color: red; text-decoration: underline; }
a:visited { color: purple; }

Web Fonts

Web Fonts allow you to use custom styles from the web (like Google Fonts) so your site looks the same for everyone.
الـ Web Fonts بتخليك تستخدم خطوط مش موجودة عند المستخدم أصلاً، وبتحملها من النت زي خطوط جوجل المشهورة.
@import url('https://fonts.googleapis.com/css2?family=Roboto');

body {
  font-family: 'Roboto', sans-serif;
}

List Styling

You can change the bullet type or replace it entirely.
بتقدر تغير شكل النقطة (Bullet) بتاعة القائمة، تخليها مربع، دايرة، أو حتى تشيلها خالص.
SQUARE
  • Item A
  • Item B
CIRCLE
  • Item A
  • Item B
NONE (أشهر استخدام)
  • Item A
  • Item B

Layout & Special Effects

Justify: Spreads text evenly.
Line Height: Crucial for readability.
Letter Spacing: Adds room for a "Cyber" look.
Justify: بيوزع الكلام بالتساوي.
Line Height: أهم حاجة عشان العين متتعبش وهي بتقرأ.
Letter Spacing: بيعمل مسافات بين الحروف وبيدي شكل شيك (Cyber Look).
p {
  text-align: justify;
  line-height: 1.8;
  letter-spacing: 1px;
}
