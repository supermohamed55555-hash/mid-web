Fonts & Text Styling

Mastering typography: How to control layout, legibility, and visual impact of text content.

What is Text Styling?

Text styling in CSS means controlling how text looks and is displayed on a webpage, including font, size, color, and spacing.
Text Styling في CSS يعني التحكم في شكل النص في الصفحة زي اختيار نوع الخط، حجمه، لونه، والمسافات بين السطور والحروف.

Font Family & Types

CSS uses 'Font Stacks' as fallbacks. If the first font isn't on the user's computer, it moves to the next.
نوع الخط (Font Family) بيتحدد بقائمة خطوط احتياطية، لو أول واحد مش موجود المتصفح بيجرب اللي بعده.
SERIF (زوائد)
The quick brown fox
SANS-SERIF (بسيط)
The quick brown fox
MONOSPACE (متساوي)
The quick brown fox
p {
  font-family: "Arial", "Verdana", sans-serif;
}

Font Size, Weight & Style

Size controls dimensions (px/rem/em), weight controls thickness (bold), and style controls slanting (italic).
حجم الخط بتقدر تتحكم فيه بوحدات زي الـ px (ثابت) أو الـ rem (نسبة للصفحة). وكمان تقدر تخليه عريض (bold) أو مائل (italic).
p {
  font-size: 20px;
  font-weight: bold;
  font-style: italic;
}

Text Decoration & Transform

Text Transform: Changes the casing of the text.
- uppercase: ALL LETTERS CAPITAL.
- lowercase: all letters small.
- capitalize: First Letter Of Each Word Capital.

Text Decoration: Adds lines to the text (underline, overline, line-through).
التحويل (Transform): بيغير حالة الحروف (كبيرة أو صغيرة).
- uppercase: بيخلي كل الحروف كابيتال.
- lowercase: بيخلي كل الحروف صغيرة.
- capitalize: بيخلي أول حرف من كل كلمة بس كابيتال.

الزخرفة (Decoration): بتضيف خطوط زي خط تحت الكلام (underline) أو شطب في النص.
UPPERCASE
this text is now loud
LOWERCASE
THIS TEXT IS NOW QUIET
CAPITALIZE
every first letter is big
h1 {
  text-transform: capitalize;
  text-decoration: underline;
}

Text Shadow

CYBER DESIGN
Add depth and glowing effects using text-shadow. (x-offset, y-offset, blur, color).
بتضيف عمق أو توهج (Glow) وضل للنص عشان يبقى بارز وشكله احترافي.
h1 {
  text-shadow: 2px 2px 10px var(--primary);
}

Alignment & Spacing

Control paragraph layout and space between letters, words, and lines for better readability.
بتقدر تحازي النص (شمال، يمين، وسط) وتتحكم في المسافات بين الحروف والكلمات وحتى السطور عشان يبقى الكلام مريح للعين.
p {
  text-align: center;
  line-height: 1.6;
  letter-spacing: 2px;
}
