CSS Styling Lists

Beyond basic bullets: Advanced control over spacing, positioning, and custom list markers.

What is Styling Lists?

Styling lists in CSS means customizing how lists look, including spacing, bullets, and layout. Lists behave like text but have unique properties.
Styling Lists في CSS يعني تغيير شكل القوائم تماماً من حيث المسافات اللي حواليها، شكل النقط والترقيم، وتنظيمها في الصفحة.

Handling List Spacing

Lists come with default margin and padding. You should normalize them for consistent design.
القوائم ليها مسافات "Margin" و "Padding" افتراضية بيديها المتصفح، وعشان شكل موقعك يبقى مظبوط لازم تعدلها بنفسك.
ul, ol {
  margin: 10px;
  padding-left: 20px;
}

li {
  line-height: 1.6;
}

List Style Position

Controls whether the bullet is inside or outside the list item's box.
بتقرر هل النقطة (Bullet) تكون "جوه" الصندوق بتاع العنصر ولا "بره" في الهامش.
OUTSIDE (Default)
  • Bullet is outside the border.
  • Text wraps cleanly.
INSIDE
  • Bullet is part of the content.
  • Text wraps below the bullet.
ul {
  list-style-position: inside;
}

The Professional Method

Instead of basic bullets, use list-style: none and apply custom backgrounds or pseudo-elements for full control.
الطريقة الاحترافية إنك تشيل النقط خالص وتستخدم صورة أو أيقونة (Background Image) عشان تتحكم في مكانها وشكلها بالملي.
  • Premium Styled Item One
  • Premium Styled Item Two
  • Premium Styled Item Three
ul { list-style: none; }
ul li {
  background-image: url("star.png");
  background-repeat: no-repeat;
  padding-left: 25px;
}

Controlling Ordered Lists

Use HTML attributes like start, reversed, and value to manipulate numbering logic.
تقدر تتحكم في الأرقام؛ تبدأ من رقم معين، أو تعكس الترتيب، أو حتى تدي قيمة محددة لعنصر واحد في النص.
<ol start="4"> ... </ol> // Starts from 4
<ol reversed> ... </ol> // Counts down
<li value="10"> ... </li> // Forces this item to be 10

Nested Lists Styling

Target nested lists specifically to change their appearance at different levels.
بتقدر تدي شكل للقايمة الأساسية وشكل تاني خالص للقايمة اللي جواها (Sub-list).
ul ul {
  list-style-type: circle;
  background: rgba(255,255,255,0.02);
}
