HTML Basics

Mastering the building blocks of the web: Elements, Attributes, and Structure.

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It tells the browser how to structure and display content on a webpage.
HTML هو اللغة الأساسية المستخدمة لإنشاء صفحات الويب، وهو اللي بيقول للمتصفح يرتب ويعرض المحتوى إزاي.

HTML Elements

HTML is made of elements. Each element is used to represent a specific part of the content like a paragraph, heading, or image.
HTML مكون من عناصر، وكل عنصر بيستخدم لعرض جزء معين من الصفحة زي نص أو عنوان.
<p>My cat is very grumpy</p>

Anatomy of an Element

An HTML element consists of an Opening tag, Content, and a Closing tag. Missing the closing tag can cause layout issues.
العنصر بيتكون من: بداية العنصر (Opening Tag)، المحتوى، ونهاية العنصر (Closing Tag). نسيان الـ closing tag ممكن يعمل مشاكل في الصفحة.

Nesting Elements

Elements can be placed inside other elements. This is called nesting. Ensure you close them in the correct order.
ممكن تحط عنصر جوه عنصر تاني، وده بنسميه Nesting. لازم تفتح وتقفل العناصر بالترتيب الصح.
<p>My cat is <strong>very</strong> grumpy</p>

Void Elements

Some elements don’t have closing tags. They are called void elements. Examples include break tags and images.
في عناصر ملهاش closing tag ودي اسمها Void Elements. زي مثلاً الـ break أو الصور.
<br>
<img src="image.jpg">

Attributes

Attributes provide extra information about elements. They always go inside the opening tag.
الـ Attributes بتضيف معلومات إضافية للعنصر، ودايماً بتتكتب في الـ opening tag.
<p class="note">Hello</p>
Common Attributes: src (image source), alt (description), width/height (size).
خصائص مشهورة: src (مصدر الصورة)، alt (وصف الصورة)، و width/height (الحجم).

Boolean Attributes

Some attributes don’t need values. If they exist, they are considered true.
في Attributes مش محتاجة قيمة، مجرد وجودها يعني إنها مفعلة (True).
<input disabled>

Links Example

<a href="https://google.com">Go to Google</a>
The href attribute defines the destination URL.
الـ href بيحدد المكان أو اللينك اللي هتروحله لما تضغط.

HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <p>This is my page</p>
  </body>
</html>
<!DOCTYPE html>: Defines HTML version.
<html>: Root element.
<head>: Contains metadata.
<title>: Page title in browser tab.
<body>: Contains visible content.
DOCTYPE: تعريف نوع الصفحة.
html: العنصر الرئيسي.
head: معلومات الصفحة (زي العنوان والـ SEO).
title: عنوان الصفحة اللي بيظهر في التاب.
body: المحتوى اللي بيظهر للمستخدم.

Whitespace & Special Characters

Whitespace: Extra spaces in code are ignored by the browser.
Special Characters: Symbols like < or > must be encoded as &lt; or &gt;.
المسافات: المسافات الكتير في الكود المتصفح بيتجاهلها.
الرموز الخاصة: في رموز لازم تكتب بطريقة خاصة زي &lt; و &gt;.

Comments

<!-- This is a comment -->
Comments are ignored by the browser and are used for documentation.
التعليقات المتصفح مش بيشوفها، بنستخدمها بس عشان نسيب ملاحظات لينا في الكود.
