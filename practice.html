Q1: Multiple Choice (20 Questions)

MCQ 1

Which tag defines the most important heading?

a) <h6> | b) <heading> | c) <h1> | d) <head>
Correct: c) <h1>
MCQ 2

Attribute for defining the destination of a link?

a) link | b) href | c) src | d) target
Correct: b) href
MCQ 3

Which is NOT a valid value for CSS display?

a) block | b) inline | c) grid | d) center
Correct: d) center (It's a value for text-align, not display)
MCQ 4

Character to represent an ID selector in CSS?

a) . | b) # | c) ! | d) @
Correct: b) #
MCQ 5

Which tag is used to create an unordered list?

a) <ol> | b) <ul> | c) <li> | d) <dl>
Correct: b) <ul>
MCQ 6

Default value of position property?

a) relative | b) fixed | c) static | d) absolute
Correct: c) static
MCQ 7

The calc() function prevents ______ fonts on small screens.

a) big | b) hardly readable | c) bright | d) bold
Correct: b) hardly readable
MCQ 8

Used to span multiple columns in a table?

a) rowspan | b) colspan | c) column-span | d) width
Correct: b) colspan
MCQ 9

Which tag is a Void Element?

a) <p> | b) <img> | c) <div> | d) <span>
Correct: b) <img> (No closing tag needed)
MCQ 10

Property to change font size?

a) font-weight | b) font-size | c) text-size | d) font-style
Correct: b) font-size
MCQ 11

Correct HTML structure order?

a) body > head | b) head > body | c) html > head > body | d) doctype > html > head > body
Correct: d) doctype > html > head > body
MCQ 12

CSS property for bold text?

a) font-style | b) font-weight | c) text-bold | d) font-variant
Correct: b) font-weight: bold;
MCQ 13

Tag for a single table row?

a) <row> | b) <tr> | c) <td> | d) <th>
Correct: b) <tr> (Table Row)
MCQ 14

Attribute for image alternate text?

a) title | b) alt | c) name | d) desc
Correct: b) alt
MCQ 15

Value for a link to open in a new tab?

a) _self | b) _parent | c) _blank | d) _top
Correct: c) _blank
MCQ 16

Unit relative to the font size of the parent?

a) rem | b) em | c) px | d) %
Correct: b) em
MCQ 17

Which is a semantic tag?

a) <div> | b) <span> | c) <article> | d) <b>
Correct: c) <article> (Gives meaning to the content)
MCQ 18

CSS property to change the mouse cursor?

a) mouse | b) cursor | c) pointer | d) icon
Correct: b) cursor
MCQ 19

Selector for all <p> inside <div>?

a) div+p | b) div p | c) div.p | d) div>p
Correct: b) div p (Descendant selector)
MCQ 20

Purpose of the Viewport Meta tag?

a) SEO | b) Security | c) Scaling on mobile | d) Loading JS
Correct: c) Scaling on mobile (Responsive design)

Q2: Tag Functions & Usage (20 Questions)

USAGE 1

<thead>

Groups the header content in a table. | بيجمع صفوف رأس الجدول.
USAGE 2

<tbody>

Groups the main data content in a table. | بيجمع محتوى الجدول الرئيسي.
USAGE 3

<meta>

Provides metadata (info about data) like charset or viewport. | بيوصف معلومات الصفحة للمتصفح.
USAGE 4

<link>

Links an external stylesheet (CSS) to the HTML. | بيربط ملف الـ CSS الخارجي.
USAGE 5

<label>

Defines a label for an input element (improves accessibility). | بيعرف اسم ووصف للحقل وبيرتبط بيه الـ ID.
USAGE 6

<footer>

Defines the footer of a page or section. | تذييل الصفحة (بيكون فيه اللينكات والكوبي رايت).
USAGE 7

<br>

Inserts a single line break in text. | بينزل سطر جديد.
USAGE 8

<strong>

Indicates important text (semantic bold). | كلام مهم جداً للمتصفح.
USAGE 9

<mark>

Highlights/marks text for reference. | بيظلل النص (هايلايتر أصفر).
USAGE 10

<code>

Defines a piece of computer code. | بيعرض كود برمجي.
USAGE 11

<dl>

Defines a description list. | قائمة للمصطلحات وتعريفاتها.
USAGE 12

<hr>

Horizontal rule: used to separate thematic content. | خط أفقي للفصل بين المواضيع.
USAGE 13

<caption>

Defines a title/caption for a table. | عنوان للجدول بالكامل.
USAGE 14

<sup>

Defines superscript text (O^2). | نص علوي (للقوى الرياضية).
USAGE 15

<sub>

Defines subscript text (O_2). | نص سفلي (للرموز الكيميائية).
USAGE 16

<nav>

Defines navigation links. | بيحتوي على لينكات التنقل الأساسية.
USAGE 17

<fieldset>

Groups related elements in a form. | بيحيط مجموعة من حقول الفورم ببرواز.
USAGE 18

<legend>

Defines a caption for <fieldset>. | عنوان لمجموعة حقول الفورم.
USAGE 19

<input type="submit">

Submits form data to the server. | زرار إرسال البيانات للسيرفر.
USAGE 20

<form action="...">

Specifies where the form data is sent. | بيحدد الصفحة اللي هيروح لها البيانات.
