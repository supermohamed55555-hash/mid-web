CSS Basics

Bringing style, life, and structure to your web pages with Cascading Style Sheets.

What is CSS?

CSS (Cascading Style Sheets) is used to style and design web pages. It controls how HTML elements look — such as colors, fonts, spacing, and layout.
CSS هي اللغة المسؤولة عن شكل وتصميم الموقع، وهي اللي بتتحكم في الألوان، الخطوط، المسافات، وشكل الصفحة العام.
HTML is used for structure (skeleton), while CSS is used for design (skin/clothes). Without CSS, pages look plain.
HTML بيبني هيكل الصفحة، لكن CSS بيخليها شكلها شيك. من غير CSS الموقع هيبقى عبارة عن نصوص بسيطة تحت بعضها.

CSS Syntax

p {
  color: blue;
  font-size: 20px;
}
Selector: The HTML element you want to style (p).
Property: What aspect you want to change (color).
Value: The specific style you want to apply (blue).
Selector: دا العنصر اللي هنغير شكله.
Property: دي الخاصية اللي هنعدلها (زي اللون).
Value: دي القيمة اللي عايزينها تظهر (زي لون أبيض مثلاً).

Ways to Add CSS

Inline

Style inside HTML tag.

<p style="color:red">
Internal

Style inside <head>.

<style>...</style>
External

Separate .css file (Best Practice).

<link rel="stylesheet">

CSS Selectors

Selectors point to the HTML elements you want to style.
الـ Selectors بتحدد بالضبط أي جزء في الصفحة انت عايز تظبط شكله.
p Element Selector
.box Class Selector
#title ID Selector

The Box Model

Every element in CSS is treated as a box. It consists of: Content, Padding, Border, and Margin.
كل عنصر في CSS بيعتبر صندوق (Box). بيتكون من: المحتوى، الحشو (Padding)، الحدود (Border)، والمسافة الخارجية (Margin).
CONTENT
Margin: Space outside the border.
Border: Line around padding and content.
Padding: Space inside the border, around content.
Margin: المسافة اللي بره حدود العنصر.
Border: الخط اللي بيفصل بين بره وجوه.
Padding: المسافة اللي بين المحتوى والحدود من جوه.

CSS Priority

CSS decides which style to apply based on Specificity (ID > Class > Element), Order, and Importance (!important).
لو فيه أكتر من ستايل لنفس الحاجة، CSS بيختار الأقوى فيهم. الـ ID أقوى من الـ Class، والـ Class أقوى من العنصر العادي.

Values and Units

pxPixels (Fixed)
%Percentage
vw/vhViewport Unit

Overflow

Overflow happens when content is too large for its container. You can hide it, scroll it, or let it show.
الـ Overflow بيحصل لما الكلام يكون كتير والمكان ضيق. بنتحكم فيه عشان ميبوظش شكل الصفحة.
