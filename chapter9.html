Responsive Web Design

One site for every device: Mastering flexibility, Fluid layouts, and Media Queries.

What is Responsive Design?

Responsive Web Design (RWD) is a way of designing websites so they work well on all devices (mobile, tablet, laptop). The goal is one site for every screen.
الـ Responsive Design هو تصميم المواقع بحيث تشتغل كويس على كل الأجهزة (موبايل، تابلت، لابتوب). الهدف إنك تعمل موقع واحد ذكي بيعرف يتأقلم لوحده على أي شاشة.

Media Queries

Media queries allow you to apply specific CSS rules based on screen width, device orientation, or resolution.
الـ Media Queries هي أهم أداة في الـ CSS، لأنها بتخليك تطبق تصميم معين بس لما الشاشة توصل لحجم انت محدده (مثلاً لو الشاشة أصغر من 768px يغير اللون).
/* Tablet & Mobile Styles */
@media (max-width: 768px) {
  body { background-color: black; }
  h1 { font-size: 1.5rem; }
}
Breakpoints: Common sizes like 600px, 768px, or 1200px where the layout shifts.
Breakpoints: دي أحجام الشاشات اللي عندها التصميم بيبدأ يتغير، زي مقاس الموبايل أو التابلت.

Mobile First Design

Start designing for the smallest screens first, then use media queries to expand the design for larger screens.
الفكر الأصح إنك تبدأ تصمم للموبايل الأول (الشاشات الصغيرة)، وبعد كدة تبدأ "تكبر" التصميم وتضيف مساحات وحاجات أكتر للأجهزة الكبيرة.

Flexible Layouts (Flex & Grid)

Use relative units (%, rem, vw) and modern layout engines like Flexbox and Grid that handle resizing automatically.
استخدم وحدات مرنة بدل الأرقام الثابتة (px). الـ Flexbox والـ Grid بيوفروا لك مرونة خرافية؛ العناصر بتتحرك وبتترتب لوحدها حسب المساحة المتاحة.

Responsive Images & Text

Images: Use max-width: 100% to prevent images from overflowing their container.
Text: Use calc() or vw units to make text size fluid.
الصور: لازم تديها عرض 100% كحد أقصى عشان متطلعش بره حدود الشاشة.
النصوص: ممكن نستخدم معادلات حسابية عشان حجم الكلام يصغر ويكبر حسب حجم الشاشة.
img { max-width: 100%; height: auto; }
h1 { font-size: calc(1.5rem + 2vw); }

The Viewport Meta Tag

Crucial for mobile devices. It tells the browser how to control the page's dimensions and scaling.
الـ Viewport Meta Tag ده أهم سطر للموبايل. من غيره، المتصفح هيعرض الموقع كأنه على كمبيوتر، وشكله هيبقى بايظ تماماً على الموبايل.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
